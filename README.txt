Instructions to begin a game:

1. Create an account.
2. If successfully created you will be directed to the login page, proceed
to login.
3. Open up a new browser window with a new profile.
4. Repeat steps 1-2 for another user in the new browser window.
5. invite the other user to play.
6. switch profiles (browser windows), accept and begin the game.
